ğŸ¨ CUSTOMIZATION GUIDE

Changing Colors

Edit the CSS variables in style.css:

```css
:root {
    --primary: #4361ee;     /* Change main brand color */
    --secondary: #3a0ca3;   /* Change secondary color */
    --accent: #f72585;      /* Change accent color */
    --light: #f8f9fa;       /* Change light background */
    --dark: #212529;        /* Change dark text color */
    /* ... */
}
```

Updating Content

1. Text Content - Edit HTML files directly
2. Images - Replace files in assets/images/
3. Icons - Use any Font Awesome icon: <i class="fas fa-icon-name"></i>
4. Links - Update href attributes in navigation and footer

Adding New Pages

1. Duplicate an existing page from pages/ directory
2. Update the page content
3. Add to navigation in all HTML files:

```html
<li><a href="pages/new-page.html">New Page</a></li>
```

Changing Fonts

Replace the font-family in style.css:

```css
* {
    font-family: 'Your Font', sans-serif;
}
```
